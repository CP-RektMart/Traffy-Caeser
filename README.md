# Traffy-Caeser

final project for DSDE

## Perequisite

- Python 3.11+
- UV

## Run local server

1. clone repository

```
git clone https://github.com/CP-RektMart/Traffy-Caeser

cd Traffy-Caeser
```

2. activate venv

```
uv venv

.venv\Scripts\activate
```

3. run docker compose

```
docker-compose up -d
```

4. start server

```
uv run main.py
```
